Class RteInfo : Mfapi.Route.RteInfo

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.Route.RteInfo()
RteInfoクラスは、ルート情報を統括するクラスです。
本クラスの下に、各ルート情報ごとのサブクラスが用意されています。
ルート検索API実行と描画実行、ルート検索API実行、ルート描画実行のメソッドを実行した際に、結果が各フィールドに格納されます。
詳細は各サブクラスの説明を参照ください。
なお、お客様側のプログラムから直接本コンストラクターを呼ばないで下さい。
（RouteクラスのrequestRouteCalcAndDraw若しくは、requestRouteCalcメソッドを呼び出した際に、 本クラスの実体である"Mfapi.RteInfo"が生成されます。）
Field Summary
Field Attributes Field Name and Description
 
calcData
ルート検索結果
ルート検索APIサーバーから返却されたJSON形式の結果をオブジェクト型に変換した結果を保持します。
ルート検索結果の出力仕様は、サーバーAPI技術仕様書 2.5.1ルート検索をご覧ください。
ルート検索に成功した場合は、statusに"success"を保持します。
ルート検索に失敗した場合や、ルート検索APIサーバーに接続できなかった場合等はエラーコードを保持します。
エラーコードはサーバーAPI技術仕様書の付則 エラーコード一覧をご覧ください。
 
calcOptions
ルート検索条件
ルート検索時の条件を保持します。
requestRouteCalcメソッド、若しくはrequestRouteCalcAndDrawを実行した際に、 calcOptionsに検索条件オブジェクトが代入されるので、お客様側から直接代入しないでください。
 
drawData
ルート描画データ
ルート描画物のデータを保持します。
requestRouteDrawメソッド、若しくはrequestRouteCalcAndDrawを実行した際に、 drawOptionsに描画条件オブジェクトが代入されるので、お客様側から直接代入しないでください。
ルート描画に成功した場合は、statusに"success"を保持します。
ルート描画に失敗した場合等はエラーコードを保持します。
エラーコードはMfapi.RteDrawDataクラスのstatus項目をご覧ください。
 
drawOptions
ルート描画条件
ルート描画時の条件を保持します。
requestRouteDrawメソッド、若しくはrequestRouteCalcAndDrawを実行した際に、 drawOptionsに描画条件オブジェクトが代入されるので、お客様側から直接代入しないでください。
なお、お客様側から直接drawOptionsに描画条件オブジェクトを代入しないでください。
