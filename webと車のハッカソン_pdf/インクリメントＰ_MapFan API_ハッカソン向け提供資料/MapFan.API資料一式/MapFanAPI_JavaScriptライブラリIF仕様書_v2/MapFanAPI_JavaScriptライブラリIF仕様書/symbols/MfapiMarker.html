Class Marker : Mfapi.Features.Marker [ ]

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.Features.Marker(options, featureId)
Markerクラスは、マーカーオブジェクトを管理するクラスです。
本クラスでは、同オブジェクトの属性の確認や変更のメソッドが用意されています。
なお、お客様側のプログラムから直接本コンストラクターを呼ばないで下さい。
（MarkerLayerクラスのメソッド"addMarker"、または、 Featureクラスのメソッド"createMarker"を実行すると、 本コンストラクターが呼び出され、マーカーオブジェクトが自動的に生成されます。）
イメージファイルのパスとマーカータイプが設定されている場合、イメージファイルのパスが優先されます。 マーカータイプ指定によりマーカーを表示させる場合、指定されたサイズは適応されません。 マーカーとポップアップキーにより紐づくポップアップがある場合、マーカーの座標がポップアップの座標に適応されます。
