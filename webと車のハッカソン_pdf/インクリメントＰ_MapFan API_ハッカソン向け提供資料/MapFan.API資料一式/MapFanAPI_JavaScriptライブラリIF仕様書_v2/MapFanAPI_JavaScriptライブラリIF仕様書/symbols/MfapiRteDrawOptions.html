Class RteDrawOptions : Mfapi.Route.RteDrawOptions

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.Route.RteDrawOptions()
RteDrawOptionsクラスは、ルート描画条件のフィールドを定義したクラスです。
Routeクラスのメソッド"requestRouteCalcAndDraw()"、 または"requestRouteDraw()"のパラメーター drawOptions の型として使用します。
なお、本クラスは Mfapiクラスの"mapCreate()"メソッド実行後に利用可能です。
また、本クラスはフィールドのみで構成されるため、Object型の変数で、本クラスと同じ名称フィールドで 記述すれば、本クラスを使わなくても、作成条件の指定ができます。

drawMarkerType
ルートのマーカー描画タイプ

ルートのポリライン描画タイプ

ポリラインデザインはDrawPolylineTypeを参照してください。
なお、本指定が省略された場合、デフォルトの緑色標準サイズの値が適用されます。
 
geometryLayerId
ジオメトリーレイヤー識別子
ルート線生成時のジオメトリーレイヤー識別子を指定します。
なお、指定するジオメトリーレイヤー識別子は既に地図に追加済みの識別子を使用します。 必須項目。
 
markerLayerId
マーカーレイヤー識別子

なお、指定するマーカーレイヤー識別子は既に地図に追加済みの識別子を使用します。
 
visible

表示 ON-OFF を指定します。表示ONでは true, 表示OFFでは false を設定します。
なお、本指定が省略された場合、デフォルトの表示ONが適用されます。
