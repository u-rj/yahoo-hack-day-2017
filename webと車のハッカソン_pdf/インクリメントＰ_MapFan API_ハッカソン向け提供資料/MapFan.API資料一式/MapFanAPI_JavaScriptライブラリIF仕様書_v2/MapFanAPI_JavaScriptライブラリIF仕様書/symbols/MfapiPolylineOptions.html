Class PolylineOptions : Mfapi.Features.PolylineOptions

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.Features.PolylineOptions()
PolylineOptionsクラスは、ポリラインタイプ図形の作成条件のフィールドを定義したクラスです。
GeometryLayerクラスのメソッド"addPolyline()"、または、 Featuresクラスのメソッド"createPolyline()"のパラメーターoptionsの型として使用します。 なお、本クラスは Mfapiクラスの"mapCreate()"メソッド実行後に利用可能です。 また、本クラスはフィールドのみで構成されるため、Object型の変数で、本クラスと同じ名称フィールドで 記述すれば、本クラスを使わなくても、作成条件の指定ができます。

lineColor

不透明度
不透明度を0～1の実数値(0=透明,1=不透明、透過しない)で指定します。
なお、本指定が省略された場合、デフォルトの1が適用されます。
 
positions
座標情報
ポリライン図形の座標情報を、緯度経度座標配列で指定します。
必須項目。
 
visible
表示 ON-OFF
表示 ON-OFF を指定します。 表示ON: string = true, 表示OFF: string = false。
なお、本指定が省略された場合、デフォルトの表示ONが適用されます。
