Class Events : Mfapi.Events

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.Events()
Eventsクラスは、MapFan APIのJSライブラリ内のプログラムイベント関連を統括するクラスです。
ここでは、地図状態変化・地図イベント・マーカーイベント・ポップアップイベントを通知するコールバック関数を指定するフィールドが用意されています。
なお、お客様側のプログラムから直接本コンストラクターを呼ばないで下さい。
（Mfapiクラスの "mapCreate()" メソッドにより、自動的に本クラスのコンストラクターが実行され、 本クラスの実体である"Mfapi.Events"が生成されます。）
また、各種フィールドの設定(代入)処理は、Mfapiクラスの "mapCreate()" メソッド終了後に行ってください。

setEnableToUseMapMouseMoveEvent(use)
地図のマウスムーブイベントの利用可否情報の設定処理
地図のマウスムーブイベントの利用可否を設定します。

{boolean} setEnableToUseMapMouseMoveEvent(use)
地図のマウスムーブイベントの利用可否情報の設定処理

本指定がない場合は、falseが設定されています。
trueを設定するとお客様の利用環境によって動作が遅くなることがあります。

Parameters:
{boolean}{use}
利用可 = true, 利用不可 = false
Returns:
{boolean} 成功 = true, 失敗 = false
