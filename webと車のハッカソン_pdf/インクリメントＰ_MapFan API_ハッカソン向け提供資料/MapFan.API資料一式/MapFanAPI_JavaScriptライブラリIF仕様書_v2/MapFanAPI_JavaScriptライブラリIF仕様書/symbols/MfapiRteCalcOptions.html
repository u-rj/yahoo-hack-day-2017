Class RteCalcOptions : Mfapi.Route.RteCalcOptions

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.Route.RteCalcOptions()
RteCalcOptionsクラスは、ルート検索条件のフィールドを定義したクラスです。
Routeクラスのメソッド"requestRouteCalcAndDraw()"、 または"requestRouteCalc()"のパラメーター calcOptions の型として使用します。
各フィールドに関する詳細な設定仕様は、サーバーAPI技術仕様書 2.5.1ルート検索をご覧ください。
なお、本クラスは Mfapiクラスの"mapCreate()"メソッド実行後に利用可能です。
また、本クラスはフィールドのみで構成されるため、Object型の変数で、本クラスと同じ名称フィールドで 記述すれば、本クラスを使わなくても、作成条件の指定ができます。
