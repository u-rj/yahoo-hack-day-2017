Class Map : Mfapi.Map

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.Map(tagName, options)
Mapクラスは、MapFan APIの地図描画クラスです。
ここでは、地図コントローラーを含む地図描画設定や、各種レイヤーの追加、削除等のメソッドが用意されています。
なお、お客様側のプログラムから直接本コンストラクターを呼ばないで下さい。
（Mfapiクラスのメソッド"mapCreate()"を実行すると、本コンストラクターが呼び出され、 地図関連の処理が自動的に実行されます。）

addGeometryLayer(layerId)
ジオメトリーレイヤー追加処理
サークル等の図形オブジェクトを表示するためのジオメトリーレイヤーを作成し、追加登録します。
本メソッドでは複数のレイヤーを作成することが可能で、レイヤー識別子で使い分けることができます。 なお、指定したレイヤー識別子のレイヤーがジオメトリーレイヤーとして存在する場合、本メソッドは失敗となります。

Returns:

{boolean} setSinglePopUp(single)
単一ポップアップ表示モード設定処理
単一ポップアップ表示モードの設定値を設定します。
単一ポップアップ表示モードをtrueで設定した場合、地図上に表示できるポップアップが１つとなります。
地図上に表示中のポップアップと別のポップアップを表示した場合、現在表示中のポップアップを閉じます。
Parameters:
{boolean}{single}
単一表示モードON = true, 単一表示モードOFF = false。必須項目。
Returns:
{boolean} 成功 = true, 失敗 = false
