Class MapOptions : Mfapi.MapOptions

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.MapOptions()
MapOptionsクラスは、地図の作成条件のフィールドを定義したクラスです。
Mfapiクラスのメソッド "mapCreate()" のパラメーター options の型として使用します。
なお、本クラスはフィールドのみで構成されるため、Object型の変数で、本クラスと同じ名称フィールドで 記述すれば、本クラスを使わなくても、作成条件の指定ができます。
