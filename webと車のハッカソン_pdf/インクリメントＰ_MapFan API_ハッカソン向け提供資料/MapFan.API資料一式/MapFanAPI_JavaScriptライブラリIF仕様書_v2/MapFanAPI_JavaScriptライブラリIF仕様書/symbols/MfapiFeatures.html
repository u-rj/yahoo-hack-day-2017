Class Features : Mfapi.Features

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.Features()
Featuresクラスは、フィーチャーオブジェクトを管理するクラスです。
フィーチャーオブジェクトは、マーカー、ポップアップ、ポリゴン等の地図以外の表示物を指します。 本クラス内のメソッドにて、フィーチャーオブジェクトの作成、および削除を行うことができます。
なお、お客様側のプログラムから直接本コンストラクターを呼ばないで下さい。
（Mapクラスの "mapCreate()" メソッドにより、自動的に本クラスのコンストラクターが実行され、 本クラスの実体である"Mfapi.Features"が生成されます。）
また、本クラス内の各種メソッドの実行は、Mapクラスの "mapCreate()" メソッド終了後に行ってください。

{boolean} createCircle(options, featureId, map)
サークルタイプの図形作成処理
サークルタイプの図形フィーチャーを作成します。 作成時に、個々の図形を識別できるように、フィーチャー識別子 featureId を指定します。 また、作成条件オブジェクト options の下の各フィールドにて、図形の作成条件を指定します。
なお、本メソッドの処理だけでは、内部的にオブジェクトが作成されるだけで、画面には表示されません。 この処理の後に、GeometryLayerクラスのメソッド "attached()" を実行して、 ジオメトリーレイヤーに紐付けることで、画面に表示されるようになります。 
// ***** createCircleメソッド プログラム記述例 *****
// 条件オブジェクト生成
var options = new Mfapi.Features.CircleOptions();
// 各種属性をセット
options.position = new Mfapi.Util.LonLat(139.767222,35.6808333);
options.radius = 1000;
// サークルタイプの図形'CIRCLE_A'を作成
Mfapi.Features.createCircle(options,'CIRCLE_A');
// ジオメトリーレイヤー'LAYER_G'への紐付け
Mfapi.Map.GeometryLayer['LAYER_G'].attached('CIRCLE_A');
Parameters:
{Mfapi.Features.CircleOptions}{options}
作成条件オブジェクト。必須項目。各属性（本オブジェクトに紐づく各フィールド） の詳細な内容や指定省略時の扱い等はMfapi.Features.CircleOptionsのクラスの説明をご覧ください。

指定半径(range)の円内で、且つ距離が短い順番で配列に格納します。 
// ***** メソッド プログラム記述例 *****
// 指定した緯度経度(lonlat)から50m範囲のマーカーを検出する。
var markers = Mfapi.Features.getMarkerInRange(lonlat, 50);
for (var i = 0; i < markers.length; i++) {
    // マーカーオブジェクト
    var marker = markers[i].marker;
    // lonlatからの距離
    var distance = markers[i].distance;
}
Parameters:
{Mfapi.Util.LonLat}{location}
基準点
{number}{range}
範囲(m)
Returns:
{object} マーカーが格納された配列と距離をプロパティとして持つオブジェクトを返却します。
配列には以下のオブジェクトが格納されます。
{'marker':マーカーオブジェクト, 'distance':距離(m)}
範囲内にマーカーがあった場合は配列を返却し、
範囲内にマーカーがなかった場合は空の配列を返却します。
パラメーターエラーの場合は、nullを返却します。
