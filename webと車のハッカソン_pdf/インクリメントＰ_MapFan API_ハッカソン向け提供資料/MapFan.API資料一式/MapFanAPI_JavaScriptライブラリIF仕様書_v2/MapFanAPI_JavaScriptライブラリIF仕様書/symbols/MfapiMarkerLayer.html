Class MarkerLayer : Mfapi.Map.MarkerLayer [ ]

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.Map.MarkerLayer(layerId, event)
MarkerLayerクラスは、マーカーを表示するレイヤークラスです。
本クラスには、マーカーレイヤーの表示制御や、同レイヤーへのマーカーの追加、削除等のメソッドが用意されています。
なお、お客様側のプログラムから直接本コンストラクターを呼ばないで下さい。
（Mapクラスのマーカーレイヤー追加のメソッド "Mfapi.Map.addMarkerLayer()" を実行すると、 自動的に本クラスのコンストラクターが実行され、 本クラスの実体である "Mfapi.Map.MarkerLayer[*]" (*=レイヤー名)が生成されます。）
