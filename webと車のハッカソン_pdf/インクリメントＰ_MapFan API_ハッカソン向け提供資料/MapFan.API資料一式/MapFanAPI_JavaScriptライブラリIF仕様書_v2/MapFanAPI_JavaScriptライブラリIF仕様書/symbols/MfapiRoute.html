Class Route : Mfapi.Route [ ]

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.Route()
Routeクラスは、ルート検索・描画を管理するクラスです。
本クラス内のメソッドにて、属性の確認や変更、ルート検索・描画、および削除を行うことができます。
なお、お客様側のプログラムから直接本コンストラクターを呼ばないで下さい。
（Mapクラスの "mapCreate()" メソッドにより、自動的に本クラスのコンストラクターが実行され、 本クラスの実体である"Mfapi.Route"が生成されます。）
また、本クラス内の各種メソッドの実行は、Mapクラスの "mapCreate()" メソッド終了後に行ってください。

requestRouteCalc(routeId, calcOptions, callbackFunc)
ルート検索API実行
本メソッド実行により、以下の処理を実行します。
  1. パラメーターのルート識別子 routeIdをインデックス(キー)としてルート情報クラスのオブジェクトを Mfapi.Routeの下に連想配列として生成し、同オブジェクトの下に、パラメーターのルート検索条件オブジェクト calcOptionsを格納します。
  2. ルート検索条件オブジェクトで指定した条件でルート検索API実行を要求します。
  3. ルート検索結果を受信したら、ルート検索結果オブジェクト calcDataとして、上記ルート情報クラスのオブジェクトの下に格納します。 ルート検索APIの処理結果に関しては、ルート検索結果オブジェクト calcDataの下のフィールド status にて確認できます。
      status ： 処理結果(string)。成功="success", 失敗=エラーコード
    ルート検索結果オブジェクト calcDataの出力仕様に関しては、サーバーAPI技術仕様書 2.5.1ルート検索をご覧ください。 また、エラーコードに関しては、サーバーAPI技術仕様書の付則 エラーコード一覧をご覧ください。
  4. もし、パラメーターのコールバック関数 callbackFuncが指定されていた場合、ルート識別子 routeIdをパラメーターとして、同コールバック関数を実行します。
なお、本処理は非同期処理でルート検索サーバーと通信を行うため、2が完了した時点でメソッド自身は処理を完了し、そこまでの処理の実行結果を返却します。
 
requestRouteCalcAndDraw(routeId, calcOptions, drawOptions, callbackFunc)
ルート検索API実行と描画実行
本メソッドはルート検索API実行と描画を一括で行ないます。
処理の流れやパラメーターの扱いに関しては、ルート検索API実行メソッドrequestRouteCalc と>ルート描画実行メソッド requestRouteDraw の項目をご覧ください。
 
requestRouteDraw(routeId, drawOptions, callbackFunc)
ルート描画実行
本メソッド実行により、以下の処理を実行します。
  1. パラメーターのルート識別子 routeIdをインデックス(キー)としてルート情報クラスのオブジェクトと、同オブジェクトの下にルート検索結果オブジェクトが存在していることを確認します。 存在していない場合、falseを返却します。
  2. ルート情報クラスのオブジェクトの下に、パラメーターのルート描画の条件のオブジェクト drawOptionsを格納します。
  3. ルート描画処理(地図上のルートのポリラインの描画、および出発地、目的地、経由地のマーカー描画)を実行し、完了したら、 生成したポリラインやマーカーのフィーチャー識別子等を格納したルート描画の下に格納します。 ルート描画の処理結果に関しては、ルート描画オブジェクト drawDataの下のフィールド status にて確認できます。
      status ： 処理結果(string)。成功="success", 失敗=エラーコード
    ルート描画オブジェクト drawDataの出力仕様およびエラーコードに関しては、Mfapi.Rote.RteDrawDataクラスをご覧ください。
  4. もし、パラメーターのコールバック関数 callbackFuncが指定されていた場合、ルート識別子 routeIdをパラメーターとして、同コールバック関数を実行します。
なお、本処理はルート検索API実行の処理に合わせ、2が完了した時点でメソッド自身は処理を完了し、そこまでの処理の実行結果を返却します。
 
setVisible(routeId, visible)
