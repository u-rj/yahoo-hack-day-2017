Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi.Features.Circle(options, featureId)
Circleクラスは、サークルタイプの図形オブジェクトを管理するクラスです。
本クラスでは、同オブジェクトの属性の確認や変更のメソッドが用意されています。
なお、お客様側のプログラムから直接本コンストラクターを呼ばないで下さい。
（GeometryLayerクラスのメソッド"addCircle"、または、 Featuresクラスのメソッド"createCircle"を実行すると、 本コンストラクターが呼び出され、サークルタイプの図形オブジェクトが自動的に生成されます。）
