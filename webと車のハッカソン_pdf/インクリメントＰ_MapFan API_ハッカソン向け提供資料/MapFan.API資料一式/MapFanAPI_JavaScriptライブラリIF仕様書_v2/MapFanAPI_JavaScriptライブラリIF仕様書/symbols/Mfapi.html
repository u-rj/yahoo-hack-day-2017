Class Mfapi

Class Summary
Constructor Attributes Constructor Name and Description
 
Mfapi()
Mfapiクラスは、MapFan APIのJSライブラリ全体を統括するクラスです。
ここでは、認証および地図初期処理の受付等のメソッドやフィールドが用意されています。
なお、お客様側のプログラムから直接本コンストラクターを呼ばないで下さい。
（JSライブラリ"mfjsapi"をscriptタグで呼び出すと、自動的に本クラスのコンストラクターが実行され、 本クラスの実体である"Mfapi"が生成されます。）
Field Summary
Field Attributes Field Name and Description
 
mapHost
地図APIサーバー名
MapFan APIサービスの地図配信APIサーバー指定に利用します。
本クラスのmapCreateメソッドを実行する前に、本フィールドにサーバードメイン名(FQDN)を代入してください。
なお、お客様がご利用いただけるサーバーはサーバーAPI技術仕様書の付則 サーバーホスト一覧をご覧ください。
 
routeHost
ルートAPIサーバー名

Routeクラスの各メソッドを実行する前に、本フィールドにサーバードメイン名(FQDN)を代入してください。
なお、お客様がご利用いただけるサーバーはサーバーAPI技術仕様書の付則 サーバーホスト一覧をご覧ください。
Method Summary
Method Attributes Method Name and Description
 
auth(authId, callbackFunc)
認証リクエスト
MapFan APIサービス利用のための認証リクエストを行います。 パラメーター authId にて認証IDを、パラメーター callbackFunc に認証処理完了を通知する コールバック関数を、ご指定ください。 なお、同コールバック関数では下記パラメータが渡されます。
　status ： 処理結果(string)。成功="success", 失敗="received data error"
（本処理は非同期処理でMapFan API 認証サーバーと通信を行います。 このため、認証APIリクエスト送信後、１度、メソッドを終了して、実行結果を戻値として返却します。 その後、認証に必要な情報を受信し、認証手続きが終了したら、お客様側で用意いただいた コールバック関数を呼び出して、お客様側のプログラムに認証完了を通知します。）
再実行することで新しい期限のアクセスキーに更新されます。
 
getAuthAccessKey()
認証アクセスキー取得処理
認証処理完了後、本メソッドを実行すると、認証アクセスキーが取得できます。
認証処理未実施、または認証処理でエラーが生じた場合、空文字列が返却されます。
 
mapCreate(tagId, options)
地図初期処理
本メソッドを実行すると、地図条件パラメーターに基づいて、地図オブジェクトを生成し、 お客様側で用意したサイト(HTML)内のDIVタグ上に地図を表示します。
なお、本メソッドは認証完了後に(本クラス内の auth メソッドの完了ではなく、 auth メソッドで指定した コールバック関数内か、同関数実行後に呼ばれる処理にて)実行するようにしてください。
 
setAuthAccessKey(key)
認証アクセスキー設定処理
本メソッドを実行すると、認証のアクセスキーがセットできます。
本メソッドは、お客様側のサーバー内で認証を行なった場合、認証アクセスキーの受け渡しにご利用できます。
